Linux 学习笔记

2021-02-09

1. Linux 概述

Linux 是一个开源的基于 POSIX 和 UNIX 的多用户、多任务、支持多线程和多核的操作系统。

2. 目录管理

2.1 目录结构

  • bin
    存放系统中所有的命令。
  • etc
    存放所有的系统管理所需要的配置文件。
  • home
    用户的主目录，每个用户都有一个自己的目录。
  • root
    具有超级权限的用户的主目录
  • usr
    很多用户的应用程序和文件都存放在这个目录下。
  • var
    存放着不断扩充的文件，比如日志文件。
  • boot
    存放这启动 Linux 的核心文件。
  • lib
    存放着系统最基本的动态链接共享库。

2.2 常用命令

ls [-al]   # 列出当前目录下的所有的文件和目录，[-a] 显示隐藏文件，[-f] 显示文件的基本信息
cd    # 切换目录，[cd ..] 回到上一级目录，[cd /] 回到根目录
pwd    # 显示当前所在的目录
mkdir [-mp]   # 创建一个目录，[-m] 配置权限，[-p] 递归创建多级目录
rmdir    # 删除一个目录，只能删除空目录

cp [-pdri] [source] [destination]   
# 复制文件或目录，-[p] 复制文件属性，-[d] 若源文档为 Link file，则复制 Link file 而非文件本身，-[r] 递归复制，[-i] 若目标已存在，询问是否覆盖

rm [-rf]   # 移除文件或目录，[-f] force 的意思，不会出现警告信息，[-r] 递归删除
mv   # 移动文件，或者修改文件名称

3. 文件管理

3.1 文件类型

  • [d]：目录
  • [-]：文件
  • [l]：Link file

3.2 权限规则

  • [r]：可读，1，4，7 表示读权限
  • [w]：可写，2，5，8 表示写权限
  • [x]：可执行，3，6，9 表示执行权限
  • [-]：没有该类权限

3.3 用户分组

  • [属主]：文件所有者
  • [属组]：文件所有者的同组用户
  • [其他]：和文件所有者没有关系的用户

3.4 常用命令

chgrp [-R] 属组名 文件名    # 更改文件属组，[-R] 递归更改

chown [-R] 属主名 文件名    # 更改文件属主

chmod [-R] xyz 文件名    # 更改权限，[x] 可读的权限，[y] 可写的权限，[z] 可执行的权限，[111 222 444] 对应位置的和代表三个位置权限，[x = 3, y = 1, z = 7] : [rwxr-x--x] 

touch    # 修改文件或者目录的时间属性， 不存在则会创建
cat    # 由第一行开始显示文件内容
tac    # 从最后一行开始显示
nl      # 显示带有行号的内容
head -n [x]   # 只看开头的 x 行
tail -n [x]   # 只看最后的 x 行
less    # 一页一页的显示文件内容，/[string] 向下搜索 [string]，?[string] 向上搜索，[空格键] 翻页

3.5 硬链接和软链接

  • 硬链接
    Linux 系统中，保存在磁盘中的文件都有一个编号，称为索引节点，硬链接即是指多个文件名指向同一索引节点，相当于 Java 中的多个引用指向同一对象，软链接则是引用的引用。
  • 软链接
    相当于 Windows 中的快捷方式，A 是 B 的软链接，删除 B，A 不能访问，但硬链接依然可以访问。

4. 磁盘管理

df [-ah]    # 显示文件系统的磁盘占用情况，[-a] 显示全部的文件系统，[-h] 以人们易读的 GB、MB、KB 等格式显示
du [-ah] [文件 or 目录]    # 显示文件或者目录的磁盘占用情况

5. 用户管理

useradd [-gm] 用户名    # 创建登录的用户名，[-g] 指定用户的属组，[-m] 在 [/home/] 下创建用户目录
userdel [-r] 用户名    # 删除用户名，[-r] 删除用户目录
usermod [] 用户名    # 修改用户名的属性
su 用户名    # 切换用户名
passwd 用户名    # 修改用户的密码
passwd [-l] 用户名    # 锁定某一用户，使其不能登录

6. 进程管理

ps [-erf]    # 查看进程，[-e] 显示所有进程，[-f] 全格式显示，[-r] 只显示正在运行中的进程
ps -ef | grep [java]    # 通过管道使用 grep 过滤查找特定的进程
kill [PID]    # 杀死进程ID = PID 的进程
pstree    # 以树状形态查看各个进程之间的关系
top [-dp]    # 实时显示各个进程的运行状态，[-d] 指定两次刷新之间的间隔，[-p] 指定 PID 来监控某个进程的状态
