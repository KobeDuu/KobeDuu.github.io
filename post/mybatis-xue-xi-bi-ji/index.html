MyBatis 学习笔记

2020-12-16

1. #{} 和 ${} 的区别

  • #{} 被解析为一条 SQL 的参数占位符 ?，通过调用 PreparedStatement 的 set 方法来赋值，可以防止 SQL 注入问题。
  • ${} 在动态 SQL 解析阶段将会进行字符串拼接，字符串不会加双引号。

2. XML映射文件中的常用标签

  • <select>、<insert>、<update>、<delete>
  • <sql>、<include> 用于 SQL 语句复用。
  • <if> 用于动态 SQL。
  • <foreach> 用于遍历集合。

3. XML标签中的属性

  • useGeneratedKeys 和 keyProperty 用于将自增主键回填到 JavaBean。
  • parameterType 用于声明参数类型，基本类型不用声明。
  • resultType 用于声明结果集类型，基本类型不用声明。

4. Dao 接口和 XML 映射文件的原理

  • XML 文件中的 <select>、<insert>、<update>、<delete>都会解析成一个 MappedStatement 对象。
  • Dao 接口的全限定名+方法名可以定位到唯一的 MappedStatement 对象，因此 Dao 接口的方法不能重载。
  • MyBatis 运行时使用 JDK 动态代理为 Dao 接口生成代理对象，代理对象会拦截 Dao 接口的方法，转而执行 MappedStatement 对象的 SQL，并将结果放回。

5. Mybatis 和 Hibernate

对 JDBC 操作数据库的过程进行封装，使得开发者只关注 SQL。

  • Mybatis 简单易学，程序员直接编写原生态 SQL。
  • Hibernate 关系映射能力更强，数据库无关性好，方便移植。
